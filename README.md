
# Assignment 5 Guidelines

This project is part of Assignment 5 and aims to implement a mobile application using the following tech stack and requirements.

## Tech Stack

- Architecture Pattern: MVVM
- UI Framework: Jetpack Compose
- Dependency Injection: Dagger
- Hilt view Model
- Remote Data Access: Retrofit
- Testing: Local and Instrumental

**Libraries:**

* Compose Toolkit
* Hilt
* Dependency Injection
* Kotlin Coroutines
* Firebase Authentication
* Testing- Mockito and Espresso 

**Unit Test cases:**
Used mockito 

Completed for User Page

**UI Test cases:**
Used espresso 

## API

The application fetches data from the following RESTful API:

Base URL: https://jsonplaceholder.typicode.com/

Endpoints:
- /posts
- /comments
- /albums
- /photos
- /todos
- /users

## UI

The UI of the application has the following Components

- Side Drawer Menu
- Listing and Detail pages for each data type (posts, comments, albums, photos, todos, users)
- Remembers list positions for each page
